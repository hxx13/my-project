# Nielsen Norman Group：企业后台可用性抽检清单（10 条）

1. 任务导向：页标题说明「做什么」，副标题说明数据来源或风险。
2. 扫描效率：表格关键列靠前；长文本截断 + `title`。
3. 一致术语：同一概念在侧栏、命令面板、表格列名中用词一致。
4. 可恢复：批量删除、同步等前有确认；可提供撤销或回收站入口。
5. 错误可见：`AdminTableShell` 展示错误与重试。
6. 进度反馈：长时间请求按钮 `disabled` + 文案变化。
7. 权限可见：无权限入口禁用并说明原因（如首页卡片）。
8. 键盘：命令面板 `Ctrl+K`；`Esc` 关闭；对话框内焦点陷阱由 Radix 处理。
9. 移动可用：侧栏抽屉化，顶栏主要操作仍可达。
10. 减少噪音：Debug 页删除重复装饰文案，保留数据来源与风险提示。
